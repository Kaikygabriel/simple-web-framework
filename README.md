# simple-web-framework

Projeto simples desenvolvido com ASP.NET Core.

## 🚀 Tecnologias

* C#
* ASP.NET Core

## ▶️ Como executar

```bash
git clone https://github.com/seu-usuario/simple-web-framework.git
cd simple-web-framework
dotnet run
```

Acesse no navegador:
https://localhost:5001
